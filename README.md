# ProterquimWeb – AA2 EV02

## Descripción
ProterquimWeb es un módulo web desarrollado en Java que permite la gestión básica de productos mediante una arquitectura MVC, utilizando Servlets, JSP, JDBC y MySQL. Este proyecto hace parte de la evidencia AA2 EV02 del programa de Análisis y Desarrollo de Software.

## Tecnologías utilizadas
- Java JDK 21
- Apache NetBeans 20
- Apache Tomcat 9
- Maven
- JSP y JSTL
- Servlets
- JDBC
- MySQL
- Git y GitHub

## Funcionalidades
- Listado de productos desde base de datos MySQL
- Inserción de productos mediante formularios HTML
- Uso de métodos GET y POST
- Conexión a base de datos mediante JDBC
- Renderizado dinámico con JSP y JSTL

## Estructura del proyecto
- `src/main/java`: lógica del negocio, DAO, servlets y conexión
- `src/main/webapp`: vistas JSP y HTML
- `pom.xml`: gestión de dependencias Maven
